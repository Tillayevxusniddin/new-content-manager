# 01 システム概要

**最終更新日：** 2026-04-14

---

## 1. プロジェクト概要

| 項目 | 詳細 |
|------|------|
| システム名 | **Book Summary App（書籍要約プラットフォーム）** |
| 目的 | 社員向けに書籍の要約コンテンツ（テキスト・音声・動画）を一元管理・提供するシステム |
| 対象ユーザー | 社員（閲覧者）およびシステム管理者 |
| プラットフォーム | レスポンシブ Web アプリケーション（モバイル＋PC） |

### プロジェクト背景

多くの企業では社員のキャリア開発のために書籍が推薦されますが、実際に読む時間を確保することは難しい現状があります。Book Summary App は、社員が書籍の重要なアイデアをテキスト・音声・動画形式で手軽に学べるプラットフォームです。管理者はAdminパネルからコンテンツを管理し、社員は既存のモバイルアプリまたはブラウザから利用します。

## 6. 主要データオブジェクト

| オブジェクト | 説明 |
|-------------|------|
| `User` | ユーザー情報（ロール付き） |
| `Category` | 書籍カテゴリ |
| `BookSummary` | 書籍要約（テキスト・音声 URL・動画 URL・カバー画像） |
| `UserProgress` | ユーザー進捗（音声/動画再生位置・読書状態） |

## 7. 非機能要件

| 項目 | 要件 |
|------|------|
| 可用性 | 平日（9:00〜22:00）99% 稼働 |
| レスポンス速度 | API レスポンスタイム 500ms 未満（通常操作） |
| セキュリティ | JWT 認証・パスワードハッシュ（bcrypt）・HTTPS 必須 |
| レスポンシブ | モバイル・PC 両方で正常表示 |
| スケーラビリティ | 初期：EC2 単体、必要に応じてスケールアップ |
| データ保護 | RDS 自動バックアップ（7日間保持） |
| メディア | S3 経由の大容量ファイル直接アップロード（Presigned URL） |
